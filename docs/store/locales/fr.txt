Don't Let YouTube Track You garde votre historique de navigation YouTube privé et sous votre contrôle -- aucun compte Google requis. Enregistrez chaque vidéo, recherche, chaîne et playlist que vous visitez dans une base de données locale qui ne quitte jamais votre appareil, sauf si vous choisissez de la synchroniser.

FONCTIONNALITÉS :
- Historique YouTube privé : Enregistre automatiquement les vidéos, recherches, chaînes, shorts et playlists que vous visitez
- Stockage 100% local : Votre historique est stocké dans une base de données SQLite locale sur votre appareil, pas dans le cloud
- Synchronisation chiffrée multi-appareils : Synchronisez optionnellement votre historique sur plusieurs appareils avec un chiffrement de bout en bout AES-256
- Aucun compte requis : Associez vos appareils avec un simple code à 6 caractères -- pas d'inscription, pas d'e-mail, pas de compte Google nécessaire
- Rechercher et parcourir : Retrouvez rapidement n'importe quelle vidéo ou chaîne dans votre historique grâce à la recherche en texte intégral, organisée par date
- Exporter et importer : Téléchargez votre historique au format JSON ou CSV. Réimportez-le à tout moment. Vos données, votre format
- Nettoyage automatique : Définissez des périodes de rétention de 30 jours à illimité. Les anciennes entrées sont automatiquement supprimées
- Fonctionne partout : Chrome, Firefox, Edge, Brave, Arc, Opera et Vivaldi

COMMENT ÇA MARCHE :
Installez l'extension et elle s'exécute discrètement en arrière-plan, détectant quand vous visitez des pages YouTube. Chaque vidéo, recherche et visite de chaîne est sauvegardée dans une base de données SQLite locale en utilisant le stockage intégré de votre navigateur. Ouvrez la fenêtre popup pour parcourir, rechercher et gérer votre historique.

Vous souhaitez synchroniser entre appareils ? Créez un groupe de synchronisation, partagez le code d'association, et votre historique se synchronise automatiquement -- chiffré avant de quitter votre navigateur avec AES-256-GCM et une clé dérivée de votre code d'association. Même le serveur de synchronisation ne peut pas lire vos données.

CONFIDENTIALITÉ PAR CONCEPTION :
Pas d'analytique. Pas de télémétrie. Pas de stockage cloud par défaut. Toutes les données synchronisées sont chiffrées de bout en bout. L'extension accède uniquement à youtube.com -- rien d'autre. Votre historique de navigation n'appartient qu'à vous.
